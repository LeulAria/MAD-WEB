# MAD sample web template

## to run follow instructions below

### Clone project locally

```
git clone https://github.com/LeulAria/MAD-WEB.git
```

```
yarn install
```

#### or

```
npm install
```

### Run App

```
yarn dev
```

#### or

```
npm run dev
```
